# VOID — Immersive Scroll Experience

A cinematic creative studio website combining **Lenis**, **Locomotive Scroll** patterns, and **Satus**-inspired design into one mind-bending scroll experience.

## Features

- 🎢 **Lenis** smooth scrolling with velocity-driven marquee
- 🚂 **Locomotive Scroll** parallax depths & viewport detection
- 🌑 **Satus**-inspired dark design system with dramatic typography
- ✨ Custom cursor, scroll progress bar, back-to-top
- 🎨 AI-generated project artwork
- 📱 Fully responsive

## Sections

1. **Hero** — Cinematic split typography with word-by-word reveal
2. **Marquee** — Dual-track text that reacts to scroll velocity
3. **Projects** — Staggered grid with parallax cards
4. **Capabilities** — Sticky sidebar with dynamic counter
5. **Visual Break** — Multi-layer parallax gradients
6. **Team** — 3D perspective card reveals
7. **Contact** — Character-by-character text animation

## Tech Stack

- Vanilla HTML / CSS / JavaScript
- [Lenis](https://github.com/darkroomengineering/lenis) (CDN)
- [GSAP + ScrollTrigger](https://gsap.com) (CDN)
- Google Fonts (Outfit, Space Grotesk, Space Mono)

## Run Locally

Just open `index.html` in a browser, or serve it:

```bash
npx serve . -l 3000
```
